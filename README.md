YouTube Analytics Dashboard
Overview

The YouTube Analytics Dashboard is a Power BI project developed to analyze video performance, audience engagement, and revenue metrics across different categories and regions. It provides a consolidated view of key performance indicators and trends to support data-driven content and marketing decisions.

Objectives
Monitor overall channel performance using key metrics
Analyze views, revenue, and subscriber growth
Evaluate engagement levels across content categories
Identify optimal upload timings and trends
Understand geographical distribution of viewership
Key Features
Key Performance Indicators (KPIs):
Total Views, Revenue (INR), Subscriber Count, Average Engagement Rate, Video Duration
Category Analysis:
Comparison of views and subscribers across categories such as Vlog, Fitness, Music, Comedy, News, and Tech
Time-Based Analysis:
Year-wise trend of views and distribution of uploads across different time slots
Geographical Insights:
View distribution by top countries and map-based visualization of audience locations
Engagement Analysis:
Average engagement rate and subscriber distribution across categories
Upload Timing Analysis:
Identification of peak time slots based on upload frequency
Subscriber Insights:
Minimum and maximum subscriber count comparison across videos
Interactive Filters:
Category and country filters for dynamic data exploration
Tools and Technologies
Power BI for dashboard development and visualization
DAX (Data Analysis Expressions) for calculated measures
Data source (Excel/CSV or dataset) for input data
Dataset Description

The dataset includes the following attributes:

Video Title / ID
Category
Views
Revenue (INR)
Subscriber Count
Engagement Rate
Upload Date and Time
Video Duration
Country
Key Insights
A significant portion of views is concentrated in specific content categories
Viewer distribution is highly influenced by geographic regions, with certain countries contributing the majority of views
Upload timing impacts content visibility and performance
Subscriber growth varies across categories, indicating differing audience preferences
Usage Instructions
Open the .pbix file using Power BI Desktop
Use category and country filters to explore specific segments
Analyze KPIs and visualizations to derive insights
